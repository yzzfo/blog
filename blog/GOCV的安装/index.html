最近看到B站上有人讲解OpenCV发现里面只要很少的代码就能实现一个人脸识别感觉还蛮有意思的于是也打算玩一玩。一开始使用python来写的OpenCV,但由于是动态语言，代码补全功能实在是不好用所以改用Golang来使用OpenCV了。

1. 编译OpenCV

  1. 安装MinGW-64 下载posix-seh类型的
  2. 安装CMake
  3. go get -u -d gocv.io/x/gocv
  4. cd $GOPATH/src/gocv.io/x/gocv 若是1.14以上则安装到了pkg文件夹中了，可以通过关闭GOMOUDLE来使其安装到src目录中，或者下载后修改win_build_opencv.cmd文件中的相关路径
  5. go run ./cmd/version/main.go测试是否成功
