Zola主题

安装主题

找到Zola的主题后，可以通过切换到themes路径将主题的仓库clone到themes文件夹内

themes和templates

templates中的内容会覆盖themes/templates中的同名内容，可用于修改主题

使用主题

由于每个主题对section的处理都有区别，所以推荐直接clone主题仓库后在该仓库内运行zola serve来预览一下主题，在查看一下config.toml以及content中的结构，确定后，迁移config.toml到自己的博客内，同时适配自己的content结构。

分类

要使用分类，需要先在config.toml中设置taxonomies，这里用于添加类目例如(software,platform) 这样就可以在page中设置page的software和platform这两个类目的信息

# config.toml
taxonomies = [
    {name = "software", feed = true},
    {name = "platform", feed = true},
]

# xxx.md
[taxonomies] 
software = ["Unity"]
platform = ["IOS"]

搜索功能

zola可以生成用于搜索的数据，通过build_search_index = true即可生成搜索用的json数据，使用时需要配合elasticlunr或者fuse.
