Yolo现在主流有v5和v8两者在使用上还是有点差别的，v8整体来看比v5现代化许多。

搭建v5

由于v8中有些知识依赖v5，所以先搭建一个v5环境。

git clone https://github.com/ultralytics/yolov5.git
cd yolov5
pip install -r requirements.txt

训练模型

经过上面的命令yolov5环境就算搭建好了，接下来就是训练模型。

  1. 编辑标签文件
    在datas目录下有很多的yaml文件，以coco.yaml文件为例，我们需要在里面定义编号以及对应的名称。
path: ../datasets/coco # dataset root dir
train: images/train
val: images/val 

# Classes
names:
  0: person
  1: bicycle
  2: car
  3: motorcycle

目录结构
-datasets
 |-coco
   |-images
     |-train/*.png
     |-val/*.png
   |-lables
     |-train/*.txt
     |-val/*.txt
-data
 |-coco.yaml

上面的yaml定义了train和val目录，yolo训练的时候会自动将images改为lables然后再里面去查找对应的标签。

  1. 标记图片
    标记图片使用labelimg
  2. 执行训练
python train.py --data data/coco.yaml --weights yolov5s.pt --img 640

使用模型

model = torch.hub.load(

搭建v8

  1. pip install ultralytics
  2. 同v5中的datasets结构一致搭建数据集，并创建coco.yaml
  3. 训练yolo train data=coco8.yaml model=yolov8n.pt epochs=10 lr0=0.01
  4. 导出onnxyolo export model=yolov8n-cls.pt format=onnx

使用模型

from ultralytics import YOLO

# Create a new YOLO model from scratch
model = YOLO("yolov8n.yaml")

# Load a pretrained YOLO model (recommended for training)
model = YOLO("yolov8n.pt")

# Train the model using the 'coco8.yaml' dataset for 3 epochs
results = model.train(data="coco8.yaml", epochs=3)

# Evaluate the model's performance on the validation set
results = model.val()

# Perform object detection on an image using the model
results = model("https://ultralytics.com/images/bus.jpg")

# Export the model to ONNX format
success = model.export(format="onnx")

在Rust中使用Yolov8模型

  参考YOLOv8-ONNXRuntime-Rust

  1. 安装onnxruntime环境，在Win11中默认的版本可能过低需要去下载新的onnxruntime
  2. 查阅文档，看个版本支持的Cuda版本，例如此时1.18.1需要Cuda 12.x，cuDNN 9.x ，下载后将dll放入rust工程根目录，同时安装对应版本的Cuda和cuDNN，两者都需要，否则可能导致onnx无法使用Cuda执行。
  3. 编写build.rs
// 将根目录下dll复制到输出目录
fn main() {
    // 获取构建输出目录
    let mut target_dir: ::std::path::PathBuf =
        ::std::env::var_os("CARGO_MANIFEST_DIR").unwrap().into();
    target_dir.push("target");
    target_dir.push(::std::env::var_os("PROFILE").unwrap());

    let out_dir = target_dir.to_str().unwrap().to_string();

    // 获取当前目录
    let current_dir = env::current_dir().expect("Failed to get current directory");

    // 迭代当前目录中的所有文件
    for entry in fs::read_dir(&current_dir).expect("Failed to read current directory") {
        let entry = entry.expect("Failed to read directory entry");
        let path = entry.path();

        // 检查文件是否为 .dll 文件
        if path.extension().and_then(|ext| ext.to_str()) == Some("dll") {
            // 生成目标路径
            let file_name = path.file_name().expect("Failed to get file name");
            let dest_path = Path::new(&out_dir).join(file_name);

            // 移动文件到目标路径
            fs::copy(&path, &dest_path).expect("Failed to move file");
            println!("cargo:rerun-if-changed={}", path.display());
        }
    }
}
