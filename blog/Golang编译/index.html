Dll导出

  1. GCC安装

在win下有Mingw，MSys2，tdm-gcc这几种方法安装，前两种由于网络以及自带库的一些问题安装起来较为费事，所以推荐采用第三种方式进行安装，安装即可在Golang中使用了

  1. 导出指令
    go build --buildmode=c-shared -o xxx.dll
  2. 代码事项
  package main

  import "C"      //必须要加进去
  import "fmt"

  //export PrintBye   //固定格式，标记这个方法要导出
  func PrintBye() {
      fmt.Println("From DLL: Bye!")
  }

  //export Sum    //固定格式，标记这个方法要导出
  func Sum(a int, b int) int {
      return a + b;
  }

  func main() {
      // Need a main function to make CGO compile package as C shared library
  }
  1. C#中使用Dll
 //In Golang 
 //func XmlToJson(c C.Char*) C.Char*

 //In CSharp
 [DllImport("json2struct", CallingConvention = CallingConvention.Cdecl)]
 static extern IntPtr XmlToJson(byte[] xml);

 //容易出错的地方
 传入的byte[] xml需要按照C的形式去构造，即最后要加一个\0元素到byte尾部，C#默认的转码都是没有\0的,会导致在C解析的过程中字符串过多的进行解析
 同样的读取时也是读到\0就停止

Build的一些参数

  1. -ldflags 'flag list'
    '-s -w': 压缩编译后的体积
    -s: 去掉符号表
    -w: 去掉调试信息，不能gdb调试了
