创建磁盘

qemu-img create -f qcow2 alpine.qcow2 8G

安装系统

qemu-system-x86_64 -m 512 -nic user -boot d -cdrom alpine-standard-3.17.2-x86_64.iso -hda alpine.qcow2 -vnc :0 -cpu kvm64,+avx -enable-kvm

从磁盘启动

qemu-system-x86_64 -m 512 -nic user -hda alpine.qcow2

VNC

加入 -vnc :0 即0.0.0.0:5900,使用VNC Viewer连接即可

启用KVM虚拟化

-enable-kvm，kvm可以直接让虚拟系统使用宿主机cpu的指令集，如果cpu支持的话开启后拥有较高的性能。若想用不支持的指令则不应该加入该选项，例如N6005不支持avx，所以要使用avx时则不能填入该选项。

直接在终端中与Qemu中的系统交互

-nographic 即可直接在运行qemu命令的终端中操作虚拟系统

配置CPU

-cpu kvm6,+avx 配置cpu，kvm6为模板包含一些基础的指令，+avx为添加avx指令
-cpu help获得更多信息
