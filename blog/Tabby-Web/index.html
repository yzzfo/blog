Tabby-Web.
Posted on 2022-11-06 :: Updated on 2022-12-05 :: Tags: Docker, Terminal

有时候通过SSH连接国外VPS的时候总是容易掉线，或者被阻断，这个时候就想通过web端来直接连接VPS在网上多方寻找后发现有几个不错的项目(webssh,tabby-web)可以用来实现这个需求，其中webssh部署十分简单，但界面稍微简陋了一些，tabby-web则拥有美观的界面，但部署的时候有不少问题需要解决，本文主要记录一下tabby-web部署过程中遇到的问题。

  1. 拉取tabby-web仓库 git clone https://github.com/Eugeny/tabby-web.git
  2. 通过docker-compose安装tabby-web
        tabby:
     build: ./tabby-web
     container_name: tabby
     restart: always
     depends_on:
     - tabbydb
     # ports:
     # - 9090:80
     networks:
     - main
     volumes:
     - ./tabby/app-dist:/app-dist #这个应该不用保存才好，保存后反而容易出现缓存问题
     environment:
     - DATABASE_URL=mysql://root:password@tabbydb/tabby
     - PORT=80
     - DEBUG=False
     - APP_DIST_STORAGE=file:///app-dist
     - DOCKERIZE_ARGS="-wait tcp://tabbydb:3306 -timeout 60s"
     - SOCIAL_AUTH_GITHUB_KEY=client_id #在github中申请新的应用
     - SOCIAL_AUTH_GITHUB_SECRET=secret

 tabbydb:
     image: mariadb:10.7.1
     container_name: tabbydb
     networks:
     - main
     restart: always
     environment:
         MARIADB_DATABASE: tabby
         MARIADB_USER: crinte
         MARIADB_PASSWORD: password
         MYSQL_ROOT_PASSWORD: password

 tabbygateway:
     image: ghcr.io/eugeny/tabby-connection-gateway:master
     container_name: tabbygateway
     networks:
     - main
     # ports:
     # - 9000
     restart: always
     environment:
         TABBY_AUTH_TOKEN: password
                                                     #如果通过类似nginx的反代且强制开启https的话这边就需要进行如下的证书设置
     command: --host 0.0.0.0 --port 443 --token-auth --certificate /letsencrypt/live/npm-23/fullchain.pem --private-key /letsencrypt/live/npm-23/privkey.pem
     volumes:
       - ./nginx/letsencrypt/:/letsencrypt
  3. 安装tabby-web应用 docker container exec tabby /manage.sh add_version 1.0.171-nightly.3 ，由于上面的配置都是拉取自最新的代码所以值1.0.171-nightly.3是根据https://app.tabby.sh中的版本号得出。
  4. 登录tabby-web并进行配置
    1. 在gateway相关的设置中输入tabby-connection-gateway地址
    2. 不同于桌面端 这里连接ssh时是直接输入 root@xxx.com 的而不是ssh root@xxx.com,后边的输入会被认为用户名是ssh root
    3. 相关的选项配置https://github.com/Eugeny/tabby-connection-gateway#all-options
