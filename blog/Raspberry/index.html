  • 挂载smaba

     sudo apt-get install cifs-utils    
 echo "//nas.me/23148625/Raspberry \
 /mnt/nas cifs auto,\
 x-systemd.automount,x-systemd.idle-timeout=30,\
 username=23148625,password=123456,\
 file_mode=0777,dir_mode=0777,\
 uid=1000,gid=1000,\
 fmask=0133,dmask=0022 0 0" >> /etc/fstab

    uid和gid的数字可以改为直接输入用户名和组名
    x-systemd.automount,x-systemd.idle-timeout=30用于自动挂载，例如访问**/mnt/nas**时将自动挂载
    fmask和file_mode之间的关系，fmask就是过滤的权限，file_mode就是本身的权限 file_mode&(!fmask)就是文件生成时所设置的权限如 0777-0222 = 0555
