  3. 添加网页登陆密码

bash下执行caddy hash-password 密码得到对应的hash值JDJhJDE0JENDbEU1ZmhPelR2RmQwNnJ2L0FldmVtODdWNG01SmR5NzExeXNBS05vWkJOdG02YzhOVVZL

在想要加密的网站中添加如下配置即可:

basicauth {
        用户名 JDJhJDE0JENDbEU1ZmhPelR2RmQwNnJ2L0FldmVtODdWNG01SmR5NzExeXNBS05vWkJOdG02YzhOVVZL
}

  1. 代理nextcloud示例
    在nextcloud的data/config/config.php中添加 'overwriteprotocol' => 'https' 即可使用https的反代来进行连接

对于http方式的连接，只需将下边的Caddy配置中的tls部分配置删除即可

--------- Caddy 配置 -------------
cloud.crinte.ml {
   file_server
   encode zstd gzip
   reverse_proxy * http://192.168.3.163:8434 {
      # header_up Host {http.reverse_proxy.upstream.hostport} #这句会导致跳转到http://192.168.3.163:8434
      header_up X-Forwarded-Host {host} #主要加上这句即可
      header_up X-Real-IP {http.request.remote.host}
      header_up X-Forwarded-Port {http.request.port}
      header_up X-Forwarded-Proto {http.request.proto}
   }
   tls /config/crinte.ga_ecc/fullchain.cer /config/crinte.ga_ecc/crinte.ga.key
}
