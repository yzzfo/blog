在Unity中翻转Mesh制作空气墙.

  有时候我们想将角色限制在一个闭合物体内部，这时我们可以通过翻转该物体的mesh达到我们期望的效果。

前置知识

  1. unity采用左手坐标系
  2. 对于一个三角面，通过左手螺旋得出的法线方向即为正面方向。
  3. 在unity默认设置的情况下，对于一个三角面来说，只有正面才有碰撞检测。

解决方法

方法一：直接在blender中翻转法线即可

方法二：通过代码翻转顶点顺序

 using System.Linq;
 
 Mesh mesh = GetComponent<MeshFilter>().mesh;
 mesh.triangles = mesh.triangles.Reverse().ToArray();

至此角色便可以在物体内部移动，在边缘时会产生正确的碰撞。

拓展

在遇到类似黑魂中的雾门时(单面墙)也可以采用类似的，区别在于雾门只需要一片mesh即可。
