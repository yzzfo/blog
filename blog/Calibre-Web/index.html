Calibre-Web是一个开源的电子书管理系统，最近重新部署到主机上用来整理本地的电子书，但在使用过程中遇到了一些小问题，这里记录一下。

无法找到数据库

之前安装的technosoft2000/calibre-web这个版本的容器，但看简介作者已经不再维护了于是使用linuxserver/calibre-web这个版本的容器，安装好后输入默认的账号admin密码

admin123进入初始化界面，此时选择数据库文件夹会出现找不到数据库这个报错，原因是该项目默认你是有一份calibre数据库的，这里可以选择去下载一个calibre桌面版用其生成metadata.db文件将其放入对应文件夹即可，或者下载一份空的metadata.db文件放入对应文件夹即可。

也可以使用如下方式生成metadata.db

docker exec -it calibre-web sh 
cd /app/calibre/bin 
calibredb restore_database --really-do-it --with-library /books #创建一个数据库
chmod a+w /books/metadata.db 
exit

上传书籍出现 database is lock

该原因是sqlite使用metadata.db文件存放在cifs文件中需要调整挂载参数添加nobrl即关闭字节锁，之后即可正常使用。
