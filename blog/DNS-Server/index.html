在之前弄穿透的时候没找到支持正则表达式的 dns 服务，后来搜了下发现 Coredns，这里记录一下

.:53 {

    forward . 8.8.8.8

    rewrite name substring prefix.host.come newhost.host.come

    log

}

rewrite name substring prefix.host.com newhost.host.com 该规则能够将查询的域名重写，例如 A.prefix.host.com -> A.newhost.host.com，这样就能将 A.newhost.host.com 的结果用于 A.prefix.host.come `

运行

/coredns -conf /etc/coredns/Corefile
