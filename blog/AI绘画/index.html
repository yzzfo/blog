  最近看着AI绘画挺有趣的加之新配了一台电脑于是就打算来尝试一下AI绘画。

使用工具

  • Stable Diffusion web UI 主要的工具，可用于生成图片以及训练模型
  • Kohya's GUI 主要用于训练lora等模型

安装注意事项

翻墙时

在Windos下使用clash翻墙安装时需要在对应bat文件中输入SET HTTPS_PROXY=http://ip:port，直接在powershell输入时不知为何不生效。

不翻墙时

可以采用更改源的方式进行安装

# 临时修改pip
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
# 修改git
git config --global url."https://gitproxy.com/github.com/".insteadof "https://github.com/"

参考

  1. git使用代理加速github下载
  2. 如何修改 Python 中的 pip 安装源

相关概念

Prompt 提示词，用于形容图片的词汇
CheckPoint 基础模型，泛化性好
Textual Inversion/Embedding 主要用于训练人物特征
Hypernetwork 主要用于训练画风
Lora 主要用于训练人物特征，效果比Textual Inversion/Embedding好
Dreambooth 主要用于处理输出，可以固定某样东西的出现
